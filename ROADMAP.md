# 🗺️ Roadmap - GZ-TRD (Tradutor Offline)

Este documento apresenta o planejamento de funcionalidades e melhorias futuras para o projeto.

## 📌 Versão 1.0 - MVP (Concluído ✅)

### Android App
- [x] Estrutura base com Clean Architecture
- [x] Interface com Jetpack Compose
- [x] Tradução de texto básica
- [x] Seleção de idiomas (10+ idiomas)
- [x] Histórico de traduções local
- [x] Troca rápida de idiomas
- [x] Tema claro/escuro

### Web App (PWA)
- [x] Interface responsiva com React
- [x] Tradução de texto
- [x] Seleção de idiomas
- [x] Histórico de traduções
- [x] Service Worker para offline
- [x] Suporte PWA (installable)

### DevOps
- [x] CI/CD com GitHub Actions
- [x] Build automático Android
- [x] Deploy automático Web
